\doxysubsection{Detailed Description}
Declaration of eigenvalue solvers using Power Method, Inverse Power Method, and QR Algorithm. 

Defines the abstract base class {\ttfamily \mbox{\hyperlink{classSolver}{Solver}}} and its derived classes for computing eigenvalues using various numerical techniques. Includes support for real and complex matrices.
