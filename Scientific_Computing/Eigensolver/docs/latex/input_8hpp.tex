\doxysubsection{Detailed Description}
Declaration of the \mbox{\hyperlink{classInput}{Input}} class for managing numerical parameters and matrices from a file. 

This header file defines the {\ttfamily \mbox{\hyperlink{classInput}{Input}}} class, which provides methods to read and store configuration parameters and matrix data for numerical computations. It supports both real and complex matrices.

Key Features\+:
\begin{DoxyItemize}
\item Reads numerical parameters like tolerance, maximum iterations, shift, and method.
\item Parses square matrices from an input file in a defined format.
\item Supports validation of parameters and matrix data. 
\end{DoxyItemize}
