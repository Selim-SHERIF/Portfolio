# Machine Learning

Applied machine learning projects spanning natural language processing, medical prediction, and infrastructure applications. The emphasis throughout is on rigorous methodology: from-scratch implementations where instructive, careful feature engineering, and systematic model evaluation.

---

## Projects

```
Machine_Learning/
├── Tweet_Sentiment_Analysis_(BERT)
├── Predicting_Cardiovascular_Disease_Using_Logistic_Regression
└── Civil_Engineering_ML_Applications
```

| Project | Description | Tools |
|---|---|---|
| [Tweet Sentiment Analysis](./Tweet_Sentiment_Analysis_(BERT)) | Sentiment classification comparing shallow baselines against fine-tuned BERT and BERTweet transformer models | PyTorch, Hugging Face |
| [Cardiovascular Disease Prediction](./Predicting_Cardiovascular_Disease_Using_Logistic_Regression) | Logistic regression implemented entirely from scratch (NumPy only) for medical risk prediction on health indicator data | Python, NumPy |
| [Civil Engineering ML Applications](./Civil_Engineering_ML_Applications) | Three applied studies from HKUST CIVL 4100R: thermal preference prediction, crack detection with ResNet-50, and electricity demand forecasting | Scikit-learn, TensorFlow, PyTorch |
