\doxysubsection{Detailed Description}
Declaration of the \mbox{\hyperlink{classOutput}{Output}} class for handling formatted output of eigenvalue solver results. 

This header file defines the {\ttfamily \mbox{\hyperlink{classOutput}{Output}}} class, which provides methods for\+:
\begin{DoxyItemize}
\item Printing styled headers and results to the console.
\item Saving results to an external file.
\item Managing output formatting for both real and complex eigenvalues.
\item Displaying input summaries and error messages.
\end{DoxyItemize}

The class leverages ANSI escape codes for styled terminal output and supports detailed formatting for matrices and vectors.
