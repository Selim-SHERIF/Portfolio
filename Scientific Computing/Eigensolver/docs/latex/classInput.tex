\doxysubsection{Detailed Description}
Class to handle numerical input parameters and matrices from a file. 

The \mbox{\hyperlink{classInput}{Input}} class provides methods to read configuration data and matrices from a file and store them for numerical computations. It supports both real and complex matrices.

\begin{DoxyNote}{Note}
The matrix must be square, and numerical parameters such as tolerance and maximum iterations have specific constraints. 
\end{DoxyNote}
