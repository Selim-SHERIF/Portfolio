\doxysubsection{Detailed Description}
Handles formatted output of eigenvalue solver results. 

The {\ttfamily \mbox{\hyperlink{classOutput}{Output}}} class provides functionality to display and save results of eigenvalue solvers. It includes methods for\+:
\begin{DoxyItemize}
\item Displaying headers and results in a styled format on the console.
\item Writing results and errors to a file.
\item Managing formatted output for real and complex eigenvalues.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
ANSI escape codes are used for terminal styling, which may not be supported in all environments. 
\end{DoxyNote}
